# 2. だくてん

- だ：`da`
- ぢ：`ji` ([[3. さしすせそ#2. dakuten|じ]])
- づ：`zu` ([[3. さしすせそ#2. dakuten|ず]])
- で：`de` [[3. The particles をでにへ#2. で]]
- ど：`do`

`ぢ` and `づ` pronounce similarly with `じ` and `ず`,  so `じ` and `ず` are used more normally than `ぢ` and `づ`.

To input `ぢ` and `づ`, type `di` and `du`.
