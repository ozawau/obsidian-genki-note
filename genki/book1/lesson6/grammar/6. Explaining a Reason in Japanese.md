# 1. Introduce

- REASON + から

(から means "because")

# 2. Examples

- 忙\[いそが\]しいですから。

(Because I'm busy.)

- 行\[い\]きますから。

(Because I'll go.)

- 好\[す\]きじゃないですから。

(Because I don't like it.)

# 3. Example Dialogue

A: 恥\[はず\]かしいですから。

(Because I'm embarrassed.)

C: やめてください。

(Please stop.)

A: じゃ、帰\[かえ\]ってもいい。

(Fine, you can go home.)

C: そうですか。

(Is that so?)

A: うん、つまらないですから。

(Yeah, because you're boring.)
